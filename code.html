HOW IT WORKS

CODE

O ColorADD é uma linguagem universal, transversal e inclusiva, desenvolvida com base nas três cores primárias mais o branco e o preto, representadas através de símbolos gráficos que, conjugados, segundo o conceito de adição de cores permite ao daltónico identificar toda a paleta de cores sempre que a cor é um fator de identificação, orientação ou escolha.

Esta linguagem torna-se num “jogo mental” simples, fácil de memorizar e aplicar em situações do dia-a-dia, possibilitando aos daltónicos, de uma forma clara a identificação das cores.
